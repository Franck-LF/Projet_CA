Installing Python packages from local folder

Sources:
https://pip.pypa.io/en/stable/cli/pip_install/
https://stackoverflow.com/questions/11091623/how-to-install-packages-offline

Install a new environment with all python packages
Create a requirements file
pip download -r requirements.txt
Download all needed packages as .whl
pip download -r requirements.txt -d packages_offline
It creates a folder “packages_offline/” with all needed packages
On another folder (or another computer) create a new environment and activate it
python -m venv venv
venv\Scripts\activate
Copy the requirements.txt and the folder “packages_offline” on the new folder then Install the libraries from it
pip install --no-index --find-links=packages_offline -r requirements.txt
--no-index → interdit à pip d'utiliser Internet
--find-links=packages_offline → pip cherche les fichiers uniquement dans ce dossier
